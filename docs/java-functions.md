# ☕ Java Functions

Utility functions for Java development.

## `runjava`

Compiles and runs a Java file from the `src/` directory.

### Usage
```bash
runjava [ClassName]
```
- `ClassName`: Default is `Main`.

---

## `makejava`

Creates a new Java file with a basic boilerplate in the `src/` directory.

### Usage
```bash
makejava [ClassName]
```
- `ClassName`: Default is `Main`.
