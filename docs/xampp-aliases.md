# 🏥 XAMPP Aliases

Manage your XAMPP installation.

## `xampp`

Alias for the XAMPP control command.

### Usage
```bash
xampp start
xampp stop
xampp restart
```

---

## Fish Functions

If you are using Fish shell, these functions are available:

- `xstart`: Start XAMPP.
- `xstop`: Stop XAMPP.
- `xrestart`: Restart XAMPP.
- `xstatus`: Check XAMPP status.
